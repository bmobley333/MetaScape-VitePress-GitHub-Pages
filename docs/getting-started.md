# Getting Started with MetaScape

Welcome to the **MetaScape** system documentation. MetaScape is a modern, automated, and agent-optimized markdown engine designed to power fast, secure, and easily maintainable static sites.

## Overview

MetaScape is engineered to integrate seamlessly with automated git workflows, agentic AI assistants, and modern CI/CD deployment pipelines. By compiling markdown files into optimized, static HTML pages, MetaScape ensures lightning-fast performance and an outstanding developer experience.

## Key Features

- **Speed**: Built on top of Vite and Vue 3 for instant hot module reloading (HMR) and optimized build outputs.
- **Agent Friendly**: Designed with straightforward markdown architectures and clean JSON/YAML frontmatter configurations.
- **Easy Deployment**: Ready for automatic building and hosting via GitHub Pages.

## Next Steps

To explore more about MetaScape, you can customize your config at `.vitepress/config.js` or start adding content under the `docs/` directory.
